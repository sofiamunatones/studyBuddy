How to Define a Table
A table in HTML is a structure set of data in rows and columns.
tr Tag
"tr" stands for table row. And this element is used at the beginning and end of every row.
th Tag
"th" stands for table heading, which creates a header cell for header information.
td Tag
"td" stands for table data. This tag creates the cells of data in the table rows.
thead Tag
The "thead" element is used to group header content in an HTML table.
tfoot Tag
The "tfoot" element is used to group footer content in an HTML table.
Caption
The caption tag is used to title a table.
Colspan vs Rowspan
Colspan is an attribute used when you want to merge multiple cells in a column. Rowspan is used when merging multiple cells in a row. Colspan extends horizontally, rowspan extends vertically.
Cell Spacing vs Cell Padding
Cell spacing is the space between each cell. To change the space between cells, use the border-spacing property in CSS. Cell padding is the space between the cell edges and the cell content. To add padding onto cells, us the padding property in CSS.
How to Define Table Borders in CSS
In CSS, table borders is defined by the "border" property.
How to Control Border Widths in CSS
To control the border width of your table, use the "border-width" property in CSS.

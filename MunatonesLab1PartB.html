DOCTYPE
The DOCTYPE statement is used as an instruction to the web browser about what version of html the page is written in
Ordered Lists
An ordered list is created using the ol element. The list could be preceded by a number (default) or alphabetical identifier.
Unordered Lists
An unordered list, which is preceded by bullet points, is created using the ul element
Description Lists
Description/definition Lists are created by enclosing the term you are describing in the dt element. Then the descriptor in the dd element
Text Editor vs Word Processor
A text editor allows additons, changes, and deletions of characters to a file. An example of a text editor is VS Code and Atom. A word processor is a formatting tool which allows one to format the text and document. An example is Microsoft Word.
Client-server Model
This model shows how the internet works. The client in this model is a computer, whether that be a phone, laptop, or PC, it requests a particular service. Then, the server which is a remote computer which provides data and access to said particular services. As long as what the client is requesting is present in the servers database it can be done
FTP
FTP stands for File Transfer Protcol, which refers to a group of rules that govern how computers transfer files from one system to another through the internent.
HTTP
HTTP stands for Hypertext Transfer Protocol, which is an application-layer protocol for transmitting media documents, like html
TCP/IP
TCP/IP stands for Transmission Control Protocol/Internet Protocol. It is a set of standardized rules that allow computers to communicate on the internet.
